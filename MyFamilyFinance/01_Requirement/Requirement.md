## Overview
1. มีรายรับเป็นจำนวนเงิน เข้ามาตอนต้นเดือน
2. แตกรายรับเข้าแต่ล่ะบัญชี ซึ่่งมีหลายบัญชี แบ่งหมดทุกครั้งไม่มีรายรับเหลือ ตัวอย่างชื่อบัญขี เช่น
	1. เงินเก็บ
	2. ค่าการศึกษา
	3. การสื่อสาร
	4. ค่ากิน
	5. ช๊อปปิ้ง
	6. น้ำมันรถ
	7. ....
3. ลงรายจ่ายแต่ล่ะบัญชี ซึ่งเป็นรายจ่ายของเดือนที่ผ่านมา
4. นำยอดมาหักลบกัน เป็นยอดคงเหลือ
5. ในแต่ล่ะบัญชี สามารถผูกกับ บช.ธนาคารได้ โดยมี ธนาคารในประเทศไทย เลขที่บัญชี และ ชื่อบัญชี ที่สามารถตั้งเองได้เพื่อไว้ใช้อ้างอิง
## Stack
1. ต้องการเป็น web เน้นมือถือ deploy บน vercel
2. DB เป็น supabase
3. **Frontend:** `Next.js` (App Router)
4.  **Language:** `TypeScript`
5. **Styling:** `Tailwind CSS` + `shadcn/ui`
